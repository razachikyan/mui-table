<h1 align="center">MUI Table</h1>

# Set Up and Get Started
- Clone the repository `git clone https://github.com/razachikyan/mui-table.git`

### Set Up

- Install necessary dependencies `npm install`
  (node verion `20.12.0`)

### Get started

- Run in developmen mode `npm start`
- Build the project `npm run build`
